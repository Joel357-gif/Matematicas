# Portafolio Digital - Matemáticas

Portafolio digital desarrollado para la materia de Matemáticas en la Universidad Central del Ecuador.

## Información Académica

- **Universidad:** Universidad Central del Ecuador
- **Facultad:** Filosofía, Letras y Ciencias de la Educación
- **Carrera:** Pedagogía en Informática
- **Semestre:** Cuarto Semestre
- **Materia:** Matemáticas
- **Año:** 2026

## Descripción

Este portafolio digital recopila las actividades, temas y reflexiones desarrolladas durante el curso de Matemáticas en el cuarto semestre de la carrera de Pedagogía en Informática.

## Estructura del Proyecto

```
PORTAFOLIO DIGITAL/
├── index.html      # Página principal
├── styles.css      # Estilos CSS
├── README.md       # Documentación
└── .gitignore      # Archivos ignorados por Git
```

## Tecnologías Utilizadas

- HTML5
- CSS3
- Diseño Responsivo

## Cómo Visualizar

Abrir el archivo `index.html` en cualquier navegador web moderno.
